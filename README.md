# 📖 About This Repository

Welcome to **120-Days-of-Python-Practice**!

This repository contains **120 Python programming problems** that I solved to strengthen my understanding of Python programming fundamentals, problem-solving techniques, logic building, and coding best practices.

The goal of this challenge was to:

* ✅ Build strong Python fundamentals
* ✅ Improve problem-solving skills
* ✅ Practice writing clean and efficient code
* ✅ Learn data structures and algorithms basics
* ✅ Gain confidence in Python programming
* ✅ Create a consistent coding habit
